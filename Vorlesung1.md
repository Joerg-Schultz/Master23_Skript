# Python 1

## Datentypen

- Integer

```python
zahl = 1
zahl2 = zahl + 1
```
- Floats
    **Gleitkommazahlen** mit `.` initiieren
```python
zahl = 1.0
zahl2 = 3 * zahl
```
- String
```python
text = "langer String"
text2 = text + "Hallo"
```
- Boolean
```python
wahr_falsch = True
wahr_falsch = False
```
Nützlich für if und while
